# Quick Start Guide

Get your first panel figure ready in 5 minutes!

## 1️⃣ Open the App

- **Local**: Download `index.html` and open in browser
- **Online**: Visit your deployed Vercel URL

## 2️⃣ Upload Your SVGs

1. Click the **"Click or drag Upload SVG files"** area
2. Select 2-4 SVG files from your computer
3. Wait for thumbnails to appear (shows labels A, B, C, etc.)

## 3️⃣ Let It Auto-Arrange

Figures automatically arrange based on:
- Their aspect ratio (tall, wide, square)
- Current layout mode (auto, row, column)
- Your alignment preference (top, center, bottom)

**Result**: Perfectly aligned panel ready in 10 seconds!

## 4️⃣ Fine-Tune (Optional)

### Adjust Layout
- Change **Alignment Mode** → figures snap realignment
- Change **Layout Mode** → figures reflow
- Adjust **Spacing** → gaps update live

### Customize Labels
- Change **Label Format** (A. / A / 1)
- Change **Label Position** (left/right/top/bottom)
- Change **Label Size** (8-32px)
- Change **Label Distance** (0-80px)

### Enable Grid
- Check **Show Grid** to see reference grid
- Helps with visual alignment

### Manual Adjustments
- **Drag** any figure to new position (auto-snaps to alignment)
- **Drag corner** to resize (preserves aspect ratio)

## 5️⃣ Export Your Panel

Choose format based on your need:

### 📄 PDF (for publications)
- Click **PDF** button
- Downloads 300 DPI high-quality PDF
- Perfect for papers and printing

### 📐 SVG (for editing)
- Click **SVG** button
- Fully editable in Adobe Illustrator, Inkscape, Figma
- Scalable to any size
- Editable text labels

### 🖼️ PNG (for presentations)
- Click **PNG** button
- Universal compatibility
- Good for PowerPoint, Google Slides, web

## ⌚ That's It!

You now have a professional panel figure ready to use!

## Common Adjustments

### "Figures don't look aligned?"
1. Try dragging one slightly toward another → auto-snap kicks in
2. Check alignment mode (Top/Center/Bottom)
3. Try different layout mode

### "Labels are too close/far?"
1. Adjust **Label Distance** slider
2. Change **Label Position** if needed
3. Changes preview in real-time

### "Want different label style?"
1. Change **Label Format** (A. vs A vs 1)
2. Adjust **Label Size** for visibility
3. Export again

### "Need to save this layout?"
1. Click **💾 Layout** button
2. Copy the JSON or download as file
3. Share with colleagues
4. Later: Click **📂 Import** to restore

## Keyboard Shortcuts (Power Users)

- **Ctrl+Z** = Undo
- **Ctrl+Y** = Redo  
- **Ctrl+S** = Export SVG
- **Ctrl+P** = Export PDF

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Upload not working | Try different SVG file / refresh page |
| Figures look pixelated | Use PDF or SVG export instead of PNG |
| Changes disappeared | Export as JSON to backup layout |
| Figures won't snap | Drag directly toward another figure |
| Labels cut off | Reduce label size or move position |
